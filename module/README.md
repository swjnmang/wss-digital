# Module - Wirtschaftsschule Digital

Ein Vercel-Projekt für interaktive Module an der Wirtschaftsschule.

## Features

- **Gamification**: Punktesystem und Abzeichen
- **Aufgaben**: Interaktive Aufgabensammlungen
- **Texte**: Theoretical content and explanations
- **Übungen**: Praktische Übungen

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Deployment auf Vercel

1. Repository mit GitHub verbinden
2. Vercel mit GitHub synchronisieren
3. Vercel wird automatisch deployieren

## Tech Stack

- React 19
- TypeScript
- Vite
- Tailwind CSS
- React Router
